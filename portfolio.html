Projekt

Några exempel på det jag har fått gjort i skolan än så länge

Väder App

Examinerande uppgift som sammanfattade hela kursen .NET ramverket, man fick jobba med asynkrona metoder för att hämta data om vädret från en hemsida genom ett API. I första delen skulle datan presenteras i konsollen, andra delen skapades ett gränssnitt med MAUI.

JavaScript

Uppgiften var att hitta en lösning åt två fäktare som hade tappat bort sina svärd och kontrollera att dom fick rätt sort innan det var löst. För att lösa det behövde jag förstå eventhandlers, hur man manipulerar HTML element med JavaScript

Butik

I början av utbildningen när vi läste Programmeringsteknik C# så fick vi göra en enkel butik som slumpar fram motiv, betyg och pris för muggar och tröjor. Med hjälp av flera klasser var för sig blev det enkelt att strukturera all kod utan att behöva stöka till i Main metoden.
